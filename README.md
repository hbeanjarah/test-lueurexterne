## Setup without Docker

```shell
git clone https://github.com/hbeanjarah/test-lueurexterne.git
cd test-lueurexterne
yarn install
yarn start
```
## Setup without Docker

```shell
git clone https://github.com/hbeanjarah/test-lueurexterne.git
cd test-lueurexterne
docker-compose up
```

navigate to your favourite browser on http://localhost:3000
